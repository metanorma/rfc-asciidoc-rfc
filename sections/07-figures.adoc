
== Figures

AsciiRFC examples (corresponding to RFC XML Figures), source code Listings, and
Literals (preformatted text) are all delimited blocks. Listings and Literals
can occur nested within Examples:

[source,asciidoc]
--
.Figure 1
====
.figure1.txt
....
Figures are only permitted to contain listings 
(sourcecode), images (artwork), or literal (artwork)

This is some ASCII Art:

 _____ ___ ____ _      _   
|  ___|_ _/ ___| | ___| |_ 
| |_   | | |  _| |/ _ \ __|
|  _|  | | |_| | |  __/ |_ 
|_|   |___\____|_|\___|\__|
....

[source,ruby]
----
def listing(node)
  result = []
  if node.parent.context != :example
    result << "<figure>" 
  end
end
----
====
--


[source,xml]
----
<figure>
  <name>Figure 1</name>
  <artwork type="ascii-art" name="figure1.txt">
    Figures are only permitted to contain listings 
    (sourcecode), images (artwork), or literal (artwork)

This is some ASCII Art:

 _____ ___ ____ _      _
|  ___|_ _/ ___| | ___| |_
| |_   | | |  _| |/ _ \ __|
|  _|  | | |_| | |  __/ |_
|_|   |___\____|_|\___|\__|</artwork>
  <sourcecode type="ruby">
    def listing(node) 
      result = [] 
      if node.parent.context != :example 
        result &lt;&lt; "&lt;figure&gt;" 
      end
    end
  </sourcecode>
</figure>
----

If an AsciiRFC Listing or Literal occurs
outside of an Example, the RFC XML converter will supply the surrounding Figure
element:

[source,asciidoc]
----
....
This is some ASCII Art:

 _____ ___ ____ _      _   
|  ___|_ _/ ___| | ___| |_ 
| |_   | | |  _| |/ _ \ __|
|  _|  | | |_| | |  __/ |_ 
|_|   |___\____|_|\___|\__|
....
----

[source,xml]
----
<figure>
  <artwork type="ascii-art">This is some ASCII Art:

 _____ ___ ____ _      _
|  ___|_ _/ ___| | ___| |_
| |_   | | |  _| |/ _ \ __|
|  _|  | | |_| | |  __/ |_
|_|   |___\____|_|\___|\__|</artwork>
</figure>
----
